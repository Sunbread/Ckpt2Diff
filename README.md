# Ckpt2Diff

This user-friendly wizard is used to convert a Stable Diffusion Model from CKPT format to Diffusers format.

## Getting Started

### Dependencies

* Python 3.9 or higher
* virtualenv or conda
* requirements.txt

### Executing

* Clone this repo
* Run ckpt2diff.py and follow instructions
